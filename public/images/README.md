## Image Guidelines

1. **Size**: 1200x630 pixels (1.91:1 ratio)
2. **Format**: JPG for photos, PNG for graphics with transparency
3. **File size**: Keep under 300KB for fast loading
4. **Text**: Minimal text, ensure it's readable at small sizes
5. **Branding**: Include Discover Milwaukee logo/colors consistently
6. **Content**: Show recognizable Milwaukee imagery

## Tools for Creating OG Images

- Canva (free templates available)
- Figma
- Adobe Express
- Placid.app (for automated generation)
